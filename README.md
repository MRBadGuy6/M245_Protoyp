# M245_Protoyp
Für einen Lösungsansatz haben wir einen Protoyp erstellt

# Lösungsansatz
Es wird ein Ideenformular eingeführt womit Mitarbeiter ihre Ideen jederzeit erfassen
können. Jede Idee wird kurz beschrieben und wird für alle sichtbar gespeichert. In
taglichen Ideenrunden schaut man sich diese Ideen an und bewerten sie.
